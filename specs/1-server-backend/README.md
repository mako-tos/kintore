# サーバサイド機能 (specs/1-server-backend)

このディレクトリにはサーバサイド機能の仕様、計画、タスクが含まれます。

主要ファイル:
- `spec.md` - 仕様書（ユーザーストーリー、要件）
- `plan.md` - 実装計画（エンドポイント、データモデル、テスト）
- `tasks.md` - 実装タスク一覧

開始手順:
1. ストレージ選定（`specs/1-server-backend/research.md` を作成）
2. `src/` 以下に API 実装を追加
3. CI を整備してコントラクトテストを有効化

このプロジェクトは内部向けサービスとして設計されています（認証: 内部ネットワーク制限）。
